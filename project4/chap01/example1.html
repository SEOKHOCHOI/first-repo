HTML 페이지의 기본 구조

<!DOCTYPE html>

HTML5 문서임을 나타낸다.

<html lang="ko"> ... </html>

HTML 문서의 영역을 나타낸다. 이 영역에는 하위 태그로 head, body 태그가 사용된다.

lang
해당 문서가 작성된 언어를 설정한다. 구글과 같은 검색 사이트에서 언어별 검색을 위한 조건에 활용된다.

<head> ... </head>

html 태그안에 작성하며, HTML 문서의 정보를 담는 영역이다. head 태그 영역에는 다음의 태그가 작성될 수 있다.

  • meta 태그
  • title 태그
  • link 태그
  • script 태그
  • base 태그

<body> ... </body>

html 태그안에 작성하며, HTML 문서의 내용(컨텐츠)를 담는 영역이다. body 태그 영역에는 대부분의 태그가 작성될 수 있다.

예제 코드


<!DOCTYPE html>
<html lang="ko">
    <head>
        문서의 정보 및 외부 스타일, 스크립트, 아이콘 정보를 작성
    </head>
    <body>
        문서의 내용을 작성
    </body>
</html>

head 태그 영역

meta 태그

meta 태그는 문서의 메타데이터 정보를 담기 위해 사용하는 태그로 charset, 작성자, 문서 설명, 키워드 등의 정보를 담을 수 있다.

syntax

<meta name="속성명" content="속성값">
<meta charset="문자셋">

meta 태그에 사용할 수 있는 표준 속성명과 속성값은 다음과 같다.

author
문서 작성자 정보를 기입하기 위한 속성명이다.
keywords
해당 문서의 콘텐츠와 관련된 키워드 목록을 쉼표로 구분하여 작성하기 위한 속성명이다.
descripton
해당 페이지의 짧은 요약 설명을 작성하기 위한 속성명이다.
