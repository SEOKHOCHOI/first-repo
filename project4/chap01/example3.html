정의(설명) 목록

ol
order list 의 약자로 순서 있는 목록을 나타내기 위해 사용하는 태그이다.
ul
unorder list 의 약자로 순서 없는 목록을 나타내기 위해 사용하는 태그이다.
dl
description list의 약자로 정의(설명) 목록을 나타내기 위해 사용하는 태그이다.
