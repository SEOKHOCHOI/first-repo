br 태그를 사용합니다.
웹 페이지 화면에서 한 줄에 전부 출력할 수 없는 긴 내용이 있는 경우 자동으로 다음 줄에 출력됩니다.
글자를 강조시킬때 strong 태그를 사용 합니다.
글자의 두께를 두껍게 할 때는 b 태그를 사용 합니다.
글자의 배경에 색을 입혀 형광펜을 그은것처럼 출력할 때 mark 태그를 사용 합니다.
글자에 밑줄을 긋기위해 u 태그를 사용합니다.
글자에 취소선을 긋기위해 s 태그를 사용합니다.
글자에 기울임을 주기위해 i 태그를 사용합니다.
글자에 강조의 의미를 주기위해 em 태그를 사용합니다.
글자를 작게을 만들기위해 small 태그를 사용합니다.

단락(paragraph)을 표기하기 위해 사용하는 p 태그, 단락안에 글자가 있기 때문에 글자관련 태그를 중첩하여 사용할 수 있다.

다른 사이트 또는 책의 글을 인용한 경우 사용하는 q 태그 인용 문구에 " " 로 감싸서 출력한다.
글에 첨자표기를 위해 사용하는 태그로 위 첨자는 sup 태그를 사용하며, 아래 첨자는 sub 태그를 사용한다.
약어를 나타내기 위해 addr 태그를 사용할 수 있으며, 약어의 설명을 나타낼 수 있는 속성을 가진다. addr 태그명도 약어로 표기 한 것이다.
참고한 웹 문서나 이미지, 포스터의 제목을 표기하기 위해 cite 태그를 사용한다.

예) 저장 단축키는 Ctrl + S 입니다.
